# 功能規劃文件

此目錄包含依題目要求「使用文字敘述規劃 schema 架構、流程」的功能設計文件。

## 文件列表

| 文件 | 功能 | 說明 |
|------|------|------|
| [notification-reminder.md](./notification-reminder.md) | 訊息提醒任務即將到期 | 通知系統 Schema 與流程設計 |
| [recurring-tasks.md](./recurring-tasks.md) | 定時重複任務 | 週期性任務模板 Schema 與流程設計 |

## 文件內容

每份文件包含：

1. **功能概述** - 功能說明與使用情境
2. **Schema 設計** - ER Diagram 與欄位說明
3. **流程設計** - Sequence Diagram / Flowchart
4. **技術選型** - 建議使用的技術與服務
5. **API 規劃** - 端點設計與 Request/Response 範例
6. **注意事項** - 實作時需考慮的重點
